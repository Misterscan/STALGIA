STALGIA Help & Instructions

How to use STALGIA:

  1. Enter a description of the music in the Prompt text area.
  2. Customize options in Advanced Configuration (optional).
  3. Click Compose to generate the MIDI composition and audio preview.
  4. Go to File > Export... in the menu bar to download your files.
  5. You can refine the result using the follow-up panel or edit the code directly.
